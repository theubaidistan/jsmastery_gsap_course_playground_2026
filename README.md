## ğŸš¦ Getting Started

### Your First Animation

```javascript
// HTML
<div class="box"></div>

// JavaScript
gsap.to(".box", {
  x: 300,
  rotation: 360,
  duration: 2,
  ease: "bounce.out"
});
```

### Timeline Example

```javascript
const tl = gsap.timeline();

tl.to(".box1", { x: 100, duration: 1 })
  .to(".box2", { y: 100, duration: 1 })
  .to(".box3", { rotation: 360, duration: 1 });
```

### ScrollTrigger Example

```javascript
gsap.registerPlugin(ScrollTrigger);

gsap.to(".panel", {
  x: 500,
  scrollTrigger: {
    trigger: ".panel",
    start: "top center",
    end: "bottom center",
    scrub: true,
    markers: true
  }
});
```
